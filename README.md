# Proyecto Integrador – Data Science con Python

## Descripción
Este proyecto integra todo el flujo de trabajo de obtención, limpieza,
transformación y organización de datos utilizando Python, NumPy y Pandas.

Se parte de la generación de datos ficticios y se incorporan múltiples fuentes
de información (CSV, Excel y tabla web), para finalmente obtener un dataset
limpio y estructurado listo para análisis.

---

- Python

- Pandas
- openpyxl
